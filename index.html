Open Source & Tools

  • Dev Tools CLI CLI / Node.js

    A command-line toolkit that eliminates repetitive dev work. Scaffold new projects from opinionated templates, set up environment variables interactively, run multi-step deploy pipelines, and manage local dev services — all from a single CLI. Built with Node.js and fully extensible via plugins.

    GitHub → Docs →
  • API Mock Server Open Source

    Lightweight local mock server for frontend developers. Define your routes, responses, and status codes in a single JSON file and get a fully running REST API in seconds — no backend needed. Supports dynamic responses, delays, and error simulation for realistic frontend testing.

    GitHub → Docs →
  • React Component Kit React

    A collection of unstyled, fully accessible React components focused purely on behavior and keyboard interactions. No CSS opinions, no design system lock-in — just solid, ARIA-compliant building blocks. Modal, dropdown, combobox, tabs, and more. Drop in your own styles and ship faster.

    GitHub → Storybook →
